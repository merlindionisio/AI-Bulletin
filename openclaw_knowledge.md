## 🗺️ PHASE 1 — PRE-PRODUCTION (Now → Week 4)


### 🎓 Learning Roadmap — Tool Priority Order

1. **Week 1–2 → Make** — Cheapest entry. Visual, beginner-friendly. Build 2–3 real automations.
2. **Week 2–3 → n8n** — More power, self-hostable, open-source. Best long-term for agency.
3. **Week 3–4 → Zapier** — Learn for client context. Many clients already use it.
4. **Month 2 → GHL** — CRM + funnel + automation in one. Core tool for retainer clients.

### 🔑 Best Starter Niches (PH Agency Targeting US/AU Clients)

- **Real Estate agents** — lead follow-up, appointment booking, drip sequences
- **Clinics / Med spas** — appointment reminders, review requests, patient reactivation
- **Local service businesses (US)** — missed call text-back, lead nurture, CRM cleanup
- **E-commerce** — abandoned cart, post-purchase, review automation

### ✅ Pre-Production Checklist

- [ ] Choose niche (lock by end of Week 1)
- [ ] Build 1 demo automation per tool (Make → n8n → Zapier)
- [ ] Define Greenlight Criteria — skills required before first client pitch
- [ ] Create simple portfolio page (public Notion or basic website)
- [ ] Set up APEX HQ in Notion (done ✅)
